CartPole Q-Learning Agent
DATA 530: Artificial Intelligence - Final Project

This project implements a Q-Learning agent to solve the CartPole-v1 environment using the Gymnasium library. It discretizes the continuous state space into bins and uses a tabular Q-learning approach with an epsilon-greedy exploration strategy.

Dependencies

To run this project, you need Python installed along with the following libraries:

gymnasium: For the CartPole-v1 environment.
numpy: For efficient array manipulation and Q-table management.
matplotlib: For plotting the learning curve.

You can install them via pip using the following command:

pip install gymnasium numpy matplotlib

How to Run the Program

Open a terminal or command prompt.
Navigate to the directory containing the cartpole_q_agent.py file.
Run the script using Python:

python cartpole_q_agent.py

Program Output & Reproducing Results

When you run the program, it will perform the following steps automatically:

Initialize the Agent: Sets up the Q-table and discretization bins.
Train the Agent: Runs for 1000 episodes:
	* It prints the progress every 100 episodes to the console (e.g., Episode 100/1000 | Avg Reward: ...).

3. Generate Plot:
	* After training, it generates a plot named training_curve.png.
	* This plot visualizes the Total Reward per Episode and includes a Moving Average (50 episodes) trendline to show the "straightening out" or 		  stabilization of the learning curve.

4. Reproducibility:
	* A random seed (42) is fixed in the code. This ensures that every time you run the script, you will get the exact same training trajectory and plot.

Key Files

cartpole_q_agent.py: The main Python source code.
training_curve.png: The output plot generated by the script.
project_report.md: The project report detailing the methodology and analysis.
